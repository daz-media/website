# Brand-Assets

Hier kannst du eigene Dateien für DAZ Media & Marketing ablegen, zum Beispiel:

- Logo-Dateien
- Favicons
- Social-Media-Bilder
- Profilbilder
- Marken-Grafiken

Dateien in `public/` sind später direkt über die Website erreichbar. Eine Datei `public/assets/brand/logo.png` kann zum Beispiel im Code als `/assets/brand/logo.png` verwendet werden.
